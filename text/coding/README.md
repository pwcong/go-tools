# Coding
A tool that can convert text to special coding.

## Usage
```

Usage: coding.exe [Options] <Source>

Options:

  -m string
        coding convert method. the optional values are as follows:
                        * unicode_encode
                        * unicode_decode
                        * url_encode
                        * url_dncode
                        * md5
                        * sha1
                        * sha256
                        * sha512
                         (default "unicode_encode")

```
